Blockly Games ist eine Serie von lehrreichen Spielen, die Programmieren lehren. Es ist für Kinder konzipiert, die noch keine Vorerfahrung mit Computern haben. Nach den Spielen werden die Spieler fähig sein, konventionelle, textbasierte Programmiersprachen zu verwenden.

Die Onlineversion von Blockly zu verwenden, ist besser, da sie folgende Zusatzfunktionen enthält:

  • Die Sprache kann wärend der Benutzung geändert werden.
  • Nutzerprogramme können via Link geteilt werden.
  • Schildkröten- und Filmprogramme können auf die Reddit-Gallerie gestellt werden.
  • Täglich können neue Änderungen in die Spiele einfließen.
